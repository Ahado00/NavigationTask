# Navigation Task
## Implement Navigation in a Jetpack Compose Application
This project demonstrates how to implement Jetpack Compose Navigation with multiple screens, a dynamic TopAppBar, and a Bottom Navigation Bar. It also includes saving and displaying user input between screens.

## Features

- Jetpack Navigation: Navigate between three screens seamlessly.
- Dynamic TopAppBar: Updates its title and back button based on the active screen.
- Bottom Navigation Bar: Provides easy access to all screens.
- Data Persistence: Saves user input from the first screen and displays it on the second screen.
- UI Enhancements: Uses Material3 components for a modern UI.
